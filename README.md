# Stitched with Love Co. — Phase 1

## Preview on your laptop
1. Unzip the downloaded folder.
2. Open the folder.
3. Double-click `index.html`.

## Upload to GitHub Pages
1. Sign in to GitHub and open your website repository.
2. Upload everything INSIDE this folder, keeping the `assets` folder intact.
3. Commit the files to the `main` branch.
4. Open **Settings → Pages**.
5. Choose **Deploy from a branch**, select **main** and **/(root)**, then save.
6. GitHub will show the live website address after publishing.

## Important form note
Because GitHub Pages is free static hosting, the request buttons open the visitor's email app with all form details addressed to kristinsturgeon22@gmail.com. Visitors must press Send. No payment, booking, date, or color is automatically approved.
